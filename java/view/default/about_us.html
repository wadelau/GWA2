品味最纯生活

BoP S.L是致力于不断研究开发纯天然成分，为人类健康提供天然营养创新科技解决方案的公司。

公司总部位于欧洲，在法国、西班牙均设有工厂与实验室。

BoP S.L联合巴塞罗那大学健康医学实验室共同研发近百种天然健康营养品，我们的研发人员有着二十多年营养与药剂开发经验，产品品牌在欧洲拥有极高的知名度和美誉度。为了满足消费者“选择最佳天然营养品的健康需求”。
