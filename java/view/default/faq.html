  • 今年很流行红色的唇妆，但是怎么画才不显得老气呢?
    质地以雾面为首选 中央向两边晕染,太油亮、过於强调唇形是NG。涂上雾面的红色唇膏后，由中央向两边晕染，用棉花棒稍微擦拭唇周，模糊唇线。
  • 大脸很烦恼，怎么化妆才显得脸小一点呢?
    要脸部L围使用比平常用的粉底深一号的粉底上妆，就能营造自然的阴影。　打造古铜美肌，底妆的立体感尤为重要。修容粉是打造小V脸不可缺少的单品。
  • 如何才能拥有想VIVI的混血儿般立体的五官?
    眉毛下方打亮眉头刷上阴影,打亮在骨头部位,从眉头开始画到鼻子侧面，涂的时候不要太重手，一点点修容粉就可以。
  • 虽然是双眼皮，但是常常看起来没精神，眼妆怎么化才自然又有神呢?
    光泽眼影+眼线,先用珠光眼影打底.用眼影笔沾取少量咖啡色系眼影，画在眼睛张开时的双眼皮摺上。眼尾画较浓，强调细细的线条感。
  • 如何打造好气色底妆?
    善用妆前乳，并搭配遮瑕膏，做好修饰功夫。Step1：依肤色需求挑选类型。紫色可明亮、粉红色增添红润，一定要选对适合自己的，不然会把 肤色弄脏喔。
  • 虽然是双眼皮，但是常常看起来没精神，眼妆怎么化才自然又有神呢?
    光泽眼影+眼线,先用珠光眼影打底.用眼影笔沾取少量咖啡色系眼影，画在眼睛张开时的双眼皮摺上。眼尾画较浓，强调细细的线条感。
